upThair Now Available!!

Mar 11th, 2013

Version 1.0

upThair is an aggregator of cloud services. Version 1.0 is focused on file management within and across multiple cloud services. Within a service, it allows users to view iOS supported file types, view and save files offline, rename, copy, move, delete, and upload files.* upThair also allows users to search for and copy files across cloud services. Additionally, users can view, start, stop, and reboot their Amazon Web Services (AWS) EC2 servers.

Supported cloud services include:

  • Amazon Web Services (AWS) S3 & EC2
  • Dropbox
  • Google Drive
  • Box
  • SkyDrive

*upThair does not support file editing capability

File Preview (iOS Native Types)

  • Adobe PDF (.pdf)
  • Images (.jpg, .tif, .gif, .png, .bmp)
  • Movies (.mov, .mp4, .m4v)
  • Music (.mp3, .m4a, .wav)
  • Microsoft Excel Spreadsheets (.xls, .xlsx)
  • Microsoft PowerPoint Documents (.ppt, .pptx)
  • Microsoft Word Documents (.doc, .docx)
  • Keynote Presentations (.key)
  • Numbers Spreadsheets (.numbers)
  • Pages Documents (.pages)
  • Text Files (.txt, .rtf, and many more)
  • Web Pages (.htm, .html)
